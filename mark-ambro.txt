    1. What is Semantic HTML? 
    Semantic html are standards used to convey information about the content inside the tags.
    2. What is HTML used for? 
    Hypertext markup language, the language that web browsers read to build a webpages.
    3. What is an attribute and where do we put it? 
    links, images, lists. In HTML, elements each have their specific attribute tags: links= <a href="#">Link</a>.
    4. What is the h1 tag used for? How many times should I use it on a page?
    H1 tag is used for the title of the page. We should use it just once.

    11. What elements can I add a class to?

    19. What is the difference between a class and an id? - Give me an example of when I might use each one
    ID is used to specify one specific element. Classes can be used to identify more than one.
    You might change the background color and font color within a class, but have the class title h1 id set to a different font/size/color.
